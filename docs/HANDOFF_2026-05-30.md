## Известные проблемы (не решено в этой сессии)

| | детали |
|---|---|
| email_finder >12h | Прогон по `master_all` (~10–12к записей) не укладывается в `TimeoutStartSec` юнита. Нужен чанкинг и/или увеличение таймаута. **Это блокирует этап 5.** |
| Прогон падает по SIGTERM | Причина неясна — проверить watchdog-логику (`crimea_watchdog.service`/`.timer`), не убивает ли он живой прогон по ложному критерию. |
| ambiguous = 2 683 | После email_finder (этап 5) часть из них получит email/сайт — после этого можно глазами пройти оставшихся без контактов и снять/подтвердить `vk_review`. |
| 2ГИС / Авито | Не решается без RU residential прокси. Не приоритет. |
| Бот №1 (stdlib) | Расширение в этой сессии не делалось — только бот №2 (главный). Дублирующие команды в №1 можно оставить как есть. |
