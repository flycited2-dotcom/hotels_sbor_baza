```
↓
telegram_notify.notify (HTML сводка + превью + файлы)
↓

```

#### 2.2 HTML-sanitize в storage.save_item
- **Файл:** `_extracted/crimea_parser/utils/storage.py` — функция `_clean()`
- **Задача:** перед записью в CSV — убрать HTML-теги и эскейпить `<>&` из всех текстовых полей (`name`, `address`, `description` если будет)
- **Реализация:** `re.sub(r"<[^>]+>", " ", val)` + `html.escape(val, quote=False)` → результат с `<` `>` `&` уже как entities, Telegram parse_mode=HTML не сломается
- **Скилл:** базовый Python regex + html stdlib

#### 2.3 Усилить `pick_address` в email_finder
- **Файл:** `_extracted/crimea_parser/parsers/email_finder.py` — функция `pick_address()`
- **Текущая проблема:** regex `(?:г\.?...|улица...|пр-т...)` хватает любую строку с «улица» в ней (включая «ШИРОКИЙ ВЫБОР для кемпинга на улице»)
- **Новая логика:**
  - Требовать минимум 2 семантических токена: `(город ИЛИ посёлок) + (ул./пр./шоссе/наб.) + (текст ≤80 символов)` в ОДНОЙ строке без переносов
  - Или сначала пробовать **Schema.org** микроразметку: `<address itemprop="address">` или JSON-LD `PostalAddress.streetAddress` + `addressLocality`
  - Fallback на улучшенный regex с lookahead на ТД/ТЦ/«дом N»
- **Скилл:** regex с lookahead/lookbehind, JSON-LD parsing

#### 2.4 Retry-helper для HTTP-парсеров
- **Новый файл:** `_extracted/crimea_parser/utils/http_retry.py`
- **API:**
  ```python
  def http_json(url, *, retries=3, backoff=(1, 3, 8), timeout=20, **kwargs) -> dict
  ```
- **Логика:** urllib3-стиль — ретрай на `URLError`, `socket.timeout`, HTTP 5xx, 429; на 4xx сразу возвращаем; backoff экспоненциальный
- **Использовать в:** `parsers/vk_groups._call()`, `parsers/osm._fetch_overpass()`, `parsers/wikidata._fetch_one()`, `parsers/wikipedia._http_json()`
- **Скилл:** urllib, exception handling, retry patterns

#### 2.5 Wikidata cache 7 дней

- **Реализация:** `output/cache/wikidata_<region>.json` с `{"fetched_at": ts, "data": [...]}`. Если age < 7 дней — берём из кэша; иначе fetch + сохраняем. Кэш переживает 99% мелких изменений Wikidata.
